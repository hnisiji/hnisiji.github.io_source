# Password Protected GitHub Pages Example

This repository serves as a live example for using [instant-lock](https://github.com/hnisiji/instant-lock) to deploy a secure static site.

## Repository Structure

*   **Source Code**: [hnisiji/hnisiji.github.io_source](https://github.com/hnisiji/hnisiji.github.io_source) (This repository)
    *   Manages the source files and GitHub Actions workflow.
*   **Published Site**: [hnisiji/hnisiji.github.io](https://github.com/hnisiji/hnisiji.github.io)
    *   Contains the encrypted output deployed automatically.
